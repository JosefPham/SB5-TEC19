SwiftBoard is an online management system with the intent of optimizing teamwork.
You will be able to manage your projects, working hours, organize current projects in boards, etc.

The system is developed as a project in the course Distribution and Integration Technologies (SB5-TEC19).
This project is part of the individual #2 handin portfolio, 20-12-2019, and is built on top of the group #1 handin portfolio, 10-11-2019.
OBS. The overall project skeleton from #1 and #2 have incomplete functions as this wasn't a priority.
The priority of #1 was to set up a project skeleton, that could be used in #2 where the focus was on non-functional requirements.
